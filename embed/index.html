How to Embed a video in HTML5

Example video:



Step 1: Find a video you'd like to embed
Typically, it's best to find a video from a popular source like YouTube or Vimeo, in this example I've used a Family Guy bit where he pretends to be a veteran from Vimeo



Step 2: Locate the "Share" button
This should be pretty easy to find, as it's usually directly underneath the video player



Step 3: Click Share and copy the embed link
This should be an <iframe> element



Step 4: Go to your code editor (I'm just using Caret from the Chrome Webstore because the GitHub editor is terrible rubbish) and paste the embed code in.
You can mess with the width, height, border, whatever you want. I would also recommend you remove the <p> tags, they just link back to Vimeo and don't look very good




Step 5: Bobs your uncle
